# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2025-11-19

### Added
- 특별 오퍼가 있는 호텔을 별(⭐) 모양 마커로 표시
- 정보창에 special_offer와 offer_period 표시 추가
- 빈 값 체크 강화 (trim() 추가)

### Changed
- 별 모양 마커 크기를 원 모양과 비슷하게 조정 (scale 0.35)

## [1.0.0] - 2025-11-18

### Added
- 다국어 지도 출시 (한국어/영어/일본어)
- 호텔 검색 기능
- 디즈니랜드까지의 거리 표시
- 인터랙티브 튜토리얼
- FHR/THC 호텔 구분 (파란색/초록색)
- 가격 정보 표시
